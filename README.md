## 📘 Projects Summary

### 🧠 Artificial Neural Networks (ANN)
- Implementation of basic feedforward neural networks  
- Used for classification and prediction problems  
- Concepts: Forward Propagation, Backpropagation  

---

### 🖼️ Convolutional Neural Networks (CNN)
- Used for image-based tasks  
- Projects include image classification and digit recognition  
- Concepts: Convolution, Pooling, Feature Extraction  

---

### 🔁 Recurrent Neural Networks (RNN)
- Used for sequential data  
- Projects include time series and text-based prediction  
- Concepts: Sequence Learning, Memory Handling  

---

### 🔄 Transfer Learning
- Using pretrained models to improve performance  
- Reduces training time and increases accuracy  
- Fine-tuning deep learning models  

---

### 💬 Natural Language Processing (NLP)
- Text-based deep learning projects  
- Sentiment analysis and classification tasks
